# CrystGLUT

OpenGL context toolkit built on Freeglut and written in Crystal

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  cryst_glut:
    github: neutrinog/cryst_glut
```

## Usage

